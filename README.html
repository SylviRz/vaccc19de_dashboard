COVID-19 Impfungsdaten des RKI

Ziel dieses Dashboards ist es den Fortschritt von COVID-19 Impfungen in Deutschland zu dokumentieren und visualisieren.

Die Impfungsdaten werden täglich vom Robert-Koch-Institut (RKI) auf dieser Seite bereitgestellt. Die hier dargestellten Daten wurden mit Hilfe des R packages vaccc19de aufbereitet und können hier heruntergeladen werden.

Bevölkerungsdaten für die Bundesländer stammen vom Statistischen Bundesamt.
